# Codeshield

**Codeshield** is a web-based pickleball court booking and venue management platform designed to streamline court reservations and venue administration. The system enables players to discover available pickleball courts, book schedules in real time, and connect with verified venue owners through a user-friendly interface.

## Features

* Online pickleball court reservation
* Real-time court availability tracking
* Venue owner registration and management
* Verified venue listings
* Booking and schedule management
* Performance statistics dashboard
* Responsive and user-friendly interface

## Technology Stack

### Frontend

* HTML5
* CSS3
* JavaScript (ES6)

### Backend

* Node.js

### Database

* MongoDB

## Project Description

Codeshield was developed to provide an efficient and convenient solution for pickleball players and venue owners. Players can easily search for courts, check availability, and make reservations online, while venue owners can manage court listings, schedules, and bookings through a centralized platform.

The system aims to improve accessibility, reduce manual booking processes, and enhance the overall experience of the growing pickleball community.
